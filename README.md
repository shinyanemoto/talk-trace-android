# nemoto-works workspace

## Purpose
AIを活用して、小さなサービス・記事・ツールを継続的に作成するための作業場所。

## Main Workflow

Idea
↓
Research
↓
Article
↓
Prototype
↓
Publish

## Directory

### 00_ideas
思いついたアイデアを保存する場所。

### 01_research
市場調査、海外トレンド調査、競合分析を保存する場所。

### 02_articles
note・ブログ記事の下書きを保存する場所。

### 03_products
実際に作成するツールやサービス。

- chrome-extension
- android-app
- web-service

### 04_prompts
AIやCodexへの指示書・テンプレート置き場。

## Policy

小さく作る。
完璧より公開。
調査だけで終わらせず、アウトプットにつなげる。
